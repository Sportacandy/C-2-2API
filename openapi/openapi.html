C-2-2[B] 共有資源状態データモデル取得

共有資源状態データモデルのデータを取得します。指定領域に含まれる、指定カテゴリの共有資源状態を取得します。

query Parameters
bbox
Array of numbers <double> (bbox) = 4 items [ items <double > [ -180 .. 180 ] ]
Example: bbox=139.681994465,35.50676814,139.681985082,35.506773265

C-2-2[B] バウンディングボックス（矩形領域座標)データモデル。「SW(経度), SW(緯度), NE(経度), NE(緯度)」

object (voxel)
Example: x=12&y=23&z=24&f=0

C-2-2[B] 空間IDデータモデル

mesh
number <int32> (mesh) [ -2147483648 .. 2147483647 ]
Example: mesh=123456

C-2-2[B] メッシュコード データモデル 1st-mesh=4digits, 2nd-mesh=6digits, 3rd-mesh=8digits

city
number <int32> (city) [ -2147483648 .. 2147483647 ]
Example: city=1203

C-2-2[B] 行政区画コード データモデル

roadName
string [ 0 .. 4096 ] characters

道路名称。道路名称絞り込み対象の場合に指定可能。

category
string
Enum: "evacuation" "mobilityhub"

共有資源カテゴリ名

keyFilter
string [ 0 .. 4096 ] characters
Example: keyFilter=class4[0-9]*

共有資源名フィルター。正規表現（PCRE）で指定。

lowerBound
number <int32> [ -2147483648 .. 2147483647 ]
Example: lowerBound=0

共有資源状態値の下限値（包含）。値が整数として解釈できない文字列の場合にのみ有効。

upperBound
number <int32> [ -2147483648 .. 2147483647 ]
Example: upperBound=1

共有資源状態値の上限値（排他）。値が整数として解釈できない文字列の場合にのみ有効。
